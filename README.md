# Touta's World - Cinematic Digital Sanctuary 🏺✨

Welcome to **Touta's World**, a high-end, cinematic web experience that merges ancient Egyptian heritage with modern digital storytelling. 

Built with **Next.js 16.2.1 (Turbopack)** and **Framer Motion**, this project transforms the "Sunlit Heirloom" design template into an immersive platform for children and explorers.

## 🏛️ Key Features

- **Cinematic Experience**: Letter-by-letter headline reveals, scroll-linked parallax imagery, and signature motion easing.
- **Storyteller-First Navigation**: A floating glassmorphism navbar and immersive sub-pages (Meet Touta, The Team, Collection).
- **Curated Artifact Gallery**: A luxury e-commerce experience with 14 unique product editions across Notebooks, Puzzles, Pins, and Coloring Books.
- **Concierge Checkout**: Simplified shopping experience via WhatsApp and Email for a personal "concierge" feel.
- **Premium Design System**: Sophisticated typography (Noto Serif/Manrope), high-contrast color roles (Sage, Terracotta, Sand), and soft, iridescent glows.

## 🚀 Getting Started

### Prerequisites

- **Node.js**: >= 20.9.0
- **npm** or **yarn**

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/touta-world.git
   cd touta-world
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## 🎨 Design System: Sunlit Heirloom

- **Typeface**: 
  - Headlines: `Noto Serif` (Editorial & Timeless)
  - Body: `Manrope` (Modern & Accessible)
- **Primary Color**: `#E27D60` (Terracotta)
- **Secondary Color**: `#C2C384` (Sage)
